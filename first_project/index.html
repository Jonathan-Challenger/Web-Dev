ATM application in Java

The aim of this project was to create a functioning ATM like app using swing which is a GUI toolkit for Java. This helped me to
develop my skills in Java specifically relating to creating apps using a GUI and to develop my OOP skills.
The source code for this project can be found here.

My focus in this project was to try to make something functional. A program that would actually work in the way any login system does.

Therefore, if someone tries to login and they have not yet created an account they will be denied. They must register an account before entering the menu.

Once the user has created an account and logged in they have access to the main menu. From here they have multiple options depending on their goals.

The Java program was connected to a SQL database where all of the relative information was stored. This database was used to verify the indentity of the users.

While using the ATM program you can add new accounts to this database and you can manipulate the balance by using the deposit or the withdraw areas of the program.
